What is Mermaid?

Mermaid is a popular text-based diagramming tool that lets you generate flowcharts, sequence diagrams, class diagrams, and more using simple syntax. It’s widely used in documentation, especially for developers, as it allows diagrams to be generated directly from the code.

What is Excalidraw?

Excalidraw, on the other hand, is a collaborative online whiteboard tool that emphasizes hand-drawn aesthetics, making it feel like you're sketching on paper. With Excalidraw, you can create quick, expressive diagrams and collaborate with others in real-time. It’s designed to make the drawing experience intuitive and flexible.
